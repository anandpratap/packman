简介

NetCDF是一种自描述数据格式。

依赖包

CURL Zlib Szip HDF5 Parallel-NetCDF

安装范例

  • 编译C、C++和Fortran接口
    • $ packman install netcdf
  • 仅编译C、Fortran接口
    • $ packman install netcdf -with_cxx=false
  • 编译并行IO接口
    • $ packman install netcdf -use_mpi=<mpich|openmpi|true>
