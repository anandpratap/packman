简介

WRF是一个数值天气模式。

依赖包

NetCDF MPI

安装范例

  • 编译串行
    • $ packman install wrf -target_dir=<放置目录> \
                      -build_type=serial -use_nest=0 \
                      -run_case=em_real -with_chem=true \
                      -compiler_set_indices=<编译选项编号>
  • 编译并行（注意使用+use_mpi选项来指定MPI库）
    • $ packman install wrf -target_dir=<放置目录> \
                      -build_type=dmpar -use_nest=0 \
                      +use_mpi=<mpich|openmpi|true> \
                      -run_case=em_real -with_chem=true \
                      -compiler_set_indices=<编译选项编号>
